# IFACI - Plataforma de Supervisão Industrial Baseada em IIoT

O IFACI é uma plataforma de supervisão industrial desenvolvida com foco em IIoT (Internet Industrial of Things), permitindo monitoramento e controle remoto de dispositivos inteligentes conectados.

O sistema realiza integração entre frontend, backend, Node-RED e dispositivos ESP32, possibilitando comunicação em tempo real com sensores e atuadores industriais.

---

# Funcionalidades

- Monitoramento de sensores
  - Temperatura
  - Umidade
  - Pressão
  - Presença

- Controle remoto de dispositivos
  - Relé
  - Travas de segurança
  - Status de conexão

- Interface HMI web
- Integração com API REST
- Comunicação com dispositivos IoT

# Como Executar o Projeto

## Backend

```bash
cd backend
npm install
npm start
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
